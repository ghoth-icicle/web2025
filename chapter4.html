1. 함수: console.log() 등의 함수를 function 함수명(){}의 구조 안에 묶어서 사용

- alert()
- console.log()
- return()


2. 스코프: 선언한 변수의 적용 범위를 가리키며, 지역 스코프, 전역 스코프 등으로 나눌수 있음

- 지역 스코프: 변수를 특정 영역에서만 사용할 수 있을 때
- 전역 스코프: 프로그램 전체에서 사용할 수 있을 때
- 블록 스코프: 중괄호({})로 둘러쌓인 영역을 가리키는 블럭의 안에서만 사용할 수 있을 떄


3. 함수 표현식: 함수를 정의하는 방법 중 한가지

- 익명 함수: 함수의 이름을 쓰는것이 아닌 변수를 이용하여 함수를 사용
- 즉시 실행 함수: 함수를 미리 정의해 놓고 쓰는 것이 아닌, 함수를 정의하면서 동시에 실행하는 함수
- 화살표 함수: 화살표(=>)를 사용하여 함수를 선언
- 콜백 함수: 다른 함수의 인수를 사용하는 함수
