1. 연산자: 숫자끼리 사칙연산 및 문자열을 서로 연결하는 등의 동작을 함께 가리키는 말

숫자형
- + : 더하기
- - : 빼기
- * : 곱하기
- / : 나누기(몫)
- % : 나누기(나머지)
- ++ : 피연산자값 1 증가
- -- : 피연산자값 1 감소


문자열
- + : 문자열끼리 연결


비교 연산자

- == : 서로의 값을 비교 후 같으면 True 다르면 False 값 반환
- === : 서로의 데이터 유형을 비교 후 같으면 True 다르면 False 값 반환
- != : 서로의 값을 비교 후 다르면 True 같으면 False 값 반환
- !== : 서로의 데이터 유형을 비교 후 다르면 True 같으면 False 값 반환
- < : 오른쪽 값이 왼쪽 값보다 크면 True 작거나 같으면 False 값 반환
- <= : 오른쪽 값이 왼쪽 값보다 크거나 같으면 True 작으면 False 값 반환
- > : 왼쪽 값이 오른쪽 값보다 크면 True 작거나 같으면 False 값 반환
- >= : 왼쪽 값이 오른쪽 값보다 크거나 같으면 True 작으면 False 값 반환


논리 연산자
- ! : 반대 값 반환
- && : 서로의 값이 True일 경우에만 True 값 반환
- || : 서로의 값 중 하나라도 True 면 True 값 반환
