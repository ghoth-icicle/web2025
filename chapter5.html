1. DOM의 기초


- DOM 트리: 웹 문서 구조를 부모와 자식 관계로 구분하여 표시한 것

2. 웹 요소 접근하기

- querySelector(): 한 가지 웹 요소에 접근할 때 쓰는 함수 - querySelectorAll(): 모든 웹 요소에 접근할 때 쓰는 함수 - @@@.innerText: 웹 브라우저 창에 보이는 내용만 가져와 보여줌 - @@@.innerHTML: 태그와 내용을 함께 가져와 보여줌 - @@@.textContent: 소스에 있는 내용을 가져와 보여줌

3. 스타일 수정하기

- 요소.style.속성명: 해당 요소에 접근하여 해당 속성명 변경 가능
