1. 입출력 명령어

- alert(): 알림 창 표시

- cornfirm(): 예 or 아니오 선택

- prompt(): 간단한 값을 입력받을 수 있는 창을 표시

- console.log(): 콘솔 창에 괄호 안의 내용을 표시함

- document.write(): 괄호 안의 내용을 웹 브라우저 화면에 표시함


2. 변수

- 변수는 let, 상수는 const
Ex) let 변수명; / const 변수명; / var 변수명;


3. 자료형

- typeof(): 괄호안에 값이나 변수를 넣으면 어떤 자료형인지 알려줌

- 숫자형: 10, 3.14 등 정수와 실수를 묶어서 '숫자형'이라 칭함

- 문자열: 작은따옴표(')나 큰따옴표(")로 묶은 데이터를 의미하며 한글, 영어, 숫자 등 전부 가능

- 논리형: 참이나 거짓 값을 표현하기 위한 유형

- null: 값이 할당되지 않았을 때 변수의 초깃값

- 객체: 여러 개의 원시 유형을 하나로 묶어 놓은 것

- 배열: 여러개의 값을 저장할 수 있는 유형으로 객체의 일종

- 심벌: 유일성을 가지는 자료형


4. 자료형 변환

- 자바스크립트는 값에 따라 자료형이 결정됨

- 직접 숫자형, 문자열 등으로 변환할 수 있음

- Number(): 숫자형으로 변환

- toString(): 문자열로 변환

- Boolean(): 논리형으로 변환
